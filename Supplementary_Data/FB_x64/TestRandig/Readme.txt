TestRandig Readme

TestRandig requires three arguments:
1) A path to an inputs file
2) A path to an output basename
3) In integer representing the desired outputs type: 0 = both, 1 = ascii grid, 2 = GeoTIFF
e.g.
TestRandig G:\SampleData\yose.input G:\SampleData\Outputs\test 0

Note: Any directory in the outputs basename must already exist, TestRAndig will not create directories

The inputs file format is documented in RandigInputsFile.pdf
